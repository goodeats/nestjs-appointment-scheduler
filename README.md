# NestJS Appointment Scheduler

Practicing NestJS with building an appointment scheduler for doctors and patients

- `User` auth has signup endpoints for types: `Doctor` and `Patient`
- `User` auth has shared signin endpoint
- `Profile` has authenticated endpoints for Get, Post, Patch

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
